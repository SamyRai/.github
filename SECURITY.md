# Security Policy for Glowing Pixels UG

## Supported Versions
Please ensure you are using the latest major versions of our internal tools. We prioritize security patches for all outward-facing services on `*.bk.glpx.pro`.

## Reporting a Vulnerability
If you discover a security vulnerability in any GLPX repository, please **do not open a public issue**. 

Instead, send an e-mail to **security@glpx.pro**. All security vulnerabilities will be promptly addressed, and we will work with you to validate and remediate the issue within our clusters.
